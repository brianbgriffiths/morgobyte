This wizard will help you connect to the Yoto API so you can access your family's players, library, and content locally.

What you'll need:

  • A Yoto account
  • Access to the Yoto Developer Portal
  • About 5 minutes

Create OAuth Application

First, you need to create an OAuth application in the Yoto Developer Portal.

  1. Go to dashboard.yoto.dev
  2. Sign in with your Yoto account
  3. Navigate to "My Applications" or "OAuth Apps"
  4. Click "Create New Application"
  5. Fill in the application details:
    • Name: Yoto Local App
    • Redirect URI: http://localhost:8000/callback
  6. Save the application
  7. Copy the Client ID and Client Secret

Enter API Credentials

Paste your Client ID and Client Secret from the Yoto Developer Portal.

Authorize Application

Click the button below to authorize this application with your Yoto account.

You'll be redirected to Yoto's authorization page where you can:

  • Review the permissions being requested
  • Log in to your Yoto account (if not already logged in)
  • Authorize the application

Note: You'll be redirected back to this page after authorization.
